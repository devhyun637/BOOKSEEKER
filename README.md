# Book Seeker
## 북트레일러 공유 및 추천 서비스

### 1. 사용한 기술
- Backend : Node JS
- Frontend : React JS
- DBMS : MySQL
- Server : AWS


### 2. 기능별 구성
- 로그인/회원가입
- 커뮤니티 
- 타임라인
- 추천
- 북트레일러 등록
